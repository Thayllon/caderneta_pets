## Como Utilizar

1. **Clonar o Repositório:**
git clone https://github.com/Thayllon/caderneta_pets.git

2. **Instalar Dependências:**
cd controle-vacinas
npm install

3. **Executar o Aplicativo:**
ng serve


4. **Acessar o Aplicativo:**
Abra o navegador e vá para `http://localhost:4200`.
